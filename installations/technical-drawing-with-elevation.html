Technical 020

An abstract technical drawing built in the browser: elevation lines, notation, and a dark field. Each pass is different — adjust parameters, generate a new composition, export a PNG.

Industrial drawing is a language of constraints: dimension, grid, the elevation line that lifts a flat idea into space.
Here that language is loosened — still legible as “technical,” but freed to wander.

Process

01

Parameters

Canvas size, complexity, line weight, and which elements appear — circles, cubes, arrows, grid, elevation.

02

Notation

The drawing behaves like a sheet: labels, structure lines, and the elevation line as a hinge between planes.

03

Sound

A shuffled pentatonic sequence; each step triggers a note from embedded Salamander grand piano samples.

04

Output

Generative passes with PNG export. The same controls never quite repeat the same drawing.

Concept

Technical drawing usually serves fabrication: exact, repeatable, accountable. This piece borrows that vocabulary — line weight, grid, annotation — without a single object to build. The pleasure is in the tension between precision and drift.

Sound ties vision to time. As the composition grows, the piano marks each step, so the drawing has rhythm as well as form. The life bar at the bottom is a simple measure of how far the current pass has run.

You can treat it as a study, a performance, or a still image to keep. The browser is the studio; nothing is rendered until you ask it to run again.
