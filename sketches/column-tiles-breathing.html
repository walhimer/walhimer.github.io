Seed controls the random tile gradients. ZIP (PNGs) uses 4× (3200px). GIF uses 2× so encoding stays responsive. WebM (single or ZIP batch) uses 4× for sharper motion. Capture FPS is a hint to the browser; many systems top out near 60 unless you use 120 Hz and a supporting browser. Batch WebM is limited to 30 clips per run (long runs = large files).
