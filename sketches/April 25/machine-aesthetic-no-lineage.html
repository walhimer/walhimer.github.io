Artist Statement

This work begins with code. The code is the origin and the authority — a living digital thing that exists in time, loops, breathes, and ends. Any subsequent form must remain faithful to what the code actually does. The chain is specific: code creates a digital loop; the loop informs an object; the object opens into a physical experience.
