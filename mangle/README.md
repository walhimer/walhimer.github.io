# Mangle (GitHub Pages)

Experimental **Tone.js** player: **Start / Pause / Stop / Record** plus **Audition** (browse a local folder of samples).

- **Player:** `/mangle/index.html`
- **Audition:** `/mangle/audition.html`

After deploy, full URLs (custom domain or `github.io`):

- `https://YOUR_DOMAIN/mangle/`
- `https://YOUR_DOMAIN/mangle/audition.html`

**Tone.js** is vendored under `vendor/` (MIT — see `vendor/TONE-LICENSE.md`). The default demo loop loads a short MP3 from `tonejs.github.io` (network required for that demo unless you load a local file).

Large sample libraries stay on your machine; this site does not host them.
